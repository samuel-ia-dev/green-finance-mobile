# ADR 001: Stack

## Status

Aceito.

## Decisão

Usar Expo + React Native + TypeScript + Firebase + React Navigation.

## Motivo

- stack compatível com mobile-first e web de apoio
- integração simples com Expo para exportação e sharing
- Firebase reduz tempo de setup para auth e tempo real

## Consequências

- dependência de serviços gerenciados do Firebase
- necessidade de regras fortes no Firestore
