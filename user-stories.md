# User Stories

1. Como usuário, quero me cadastrar para acessar meu painel financeiro.
Critérios:
- consigo criar conta com email e senha
- sessão persiste após login
- erros são tratados com mensagens amigáveis

2. Como usuário, quero registrar receitas e despesas.
Critérios:
- consigo cadastrar transação
- consigo editar/excluir
- o histórico atualiza automaticamente

3. Como usuário, quero marcar uma despesa como recorrente.
Critérios:
- consigo escolher frequência
- o sistema cria ocorrências futuras
- o histórico mostra ícone 🔁
- o dashboard mostra total recorrente do mês

4. Como usuário, quero acompanhar metas.
Critérios:
- consigo criar e atualizar metas
- vejo progresso visual

5. Como usuário, quero visualizar relatórios.
Critérios:
- vejo gráficos
- recebo insights automáticos

6. Como usuário, quero exportar meus dados.
Critérios:
- consigo gerar CSV
- consigo gerar PDF
